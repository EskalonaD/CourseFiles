# Events

New Subdirectory
----------------

```
$ mkdir Events
$ cd Events
$ subl events.js
```

Set Interval
------------

```
node events.js
new visitor
new visitor
new visitor
new visitor
new visitor
new visitor
...
```

HINT: CTRL+C to exit node process

Emit Event
----------

```
$ node events02.js
new visitor
new visitor
new visitor
new visitor
new visitor
new visitor
...
```

Multiple Listeners
------------------

```
$ node events03.js
new visitor
TOTAL COUNT: 1
new visitor
TOTAL COUNT: 2
new visitor
TOTAL COUNT: 3
new visitor
TOTAL COUNT: 4
new visitor
```
