# Node vs Express

Node.js Webserver
-----------------

```
$ node web.js
```

Installing Express
------------------

```
$ npm install express
```

Express Webserver
-----------------

```
$ node app.js
```

HINT: Only one server can run at a time, hit CTRL+C to exit a running server
